# About this project

## What is this?

This is a project that is tested for and uses a Raspberry Pi 5*  to compute and process python code, and uses the Freenove 4WD Smart Car Kit for Raspberry Pi for the motherboard(FNK0043)**(SEE DISCLAIMER).
